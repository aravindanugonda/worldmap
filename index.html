True Country Size Explorer

Drag countries to compare size in different places. Overlays are latitude-corrected, and map guides show Equator/Tropics plus horizontal latitude lines.

Select a country to view details.

How to use

  1. Pick a country from the dropdown (or click one on the map).
  2. Drag the dashed outline onto another region.
  3. Use mouse-wheel / trackpad to zoom and drag the background to pan.
  4. Use Equator/Tropics/latitude guides for cleaner comparison by latitude.
