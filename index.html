True Country Size Explorer

This map uses the Equal Earth projection, which preserves area so countries are shown in size proportions that match reality better than Mercator. Select a country and drag its outline on top of others to compare scale directly.

How to use

  1. Click a country or choose it from the dropdown.
  2. Drag the highlighted outline and place it on another country.
  3. Repeat with multiple countries to compare true area visually.
