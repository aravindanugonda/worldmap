True Country Size Explorer

The base map is Mercator (like most web maps), but draggable overlays are latitude-corrected while you move them north/south. This gives a much truer visual size comparison between countries.

How to use

  1. Pick a country from the dropdown (or click one on the map).
  2. Drag the dashed outline onto another region.
  3. As you drag north/south, overlay scale updates automatically to compensate for Mercator distortion.

Select a country to view details.
